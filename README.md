## 📖 About the Project

**The Daily Blog** is a full-stack blogging platform that allows users to create, edit, and share blog posts with ease. Designed with a clean, responsive interface, it provides an enjoyable experience for both writers and readers across all devices.

---

## ✨ Features

- 📝 Create, edit, and delete blog posts.
- ❤️ Like or 👎 Dislike blog posts.
- 🏷️ Organize posts using categories and tags.
- 📱 Fully responsive design for desktop and mobile.
- 🎨 Clean and intuitive user interface.
- ⚡ Fast and modern full-stack architecture.
